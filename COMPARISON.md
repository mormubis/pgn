# Comparison

| Library                                                                      | Description                                                 |
| ---------------------------------------------------------------------------- | ----------------------------------------------------------- |
| [`pgn-parser`](https://www.npmjs.com/package/pgn-parser)                     | PEG.js-based PGN parser — JS data structures, no serializer |
| [`chess.js`](https://www.npmjs.com/package/chess.js)                         | full chess engine — PGN parsing bundled, not standalone     |
| [`chessops`](https://www.npmjs.com/package/chessops)                         | async streaming PGN parser with game tree model             |
| [`@chess-fu/pgn-parser`](https://www.npmjs.com/package/@chess-fu/pgn-parser) | PGN parser with SAN/LAN support                             |
